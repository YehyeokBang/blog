# 안녕하세요! 반가워요.

이 블로그는 **Next.js 15**의 App Router와 **Tailwind CSS v4**를 사용하여 빌드된 완전 정적(Static Export) 블로그입니다.

## 코드 하이라이팅 테스트

아래는 Shiki (`rehype-pretty-code`) 하이라이팅이 잘 적용되는지 검증하기 위한 코드 블록입니다.

```typescript
// Hello World 함수
function greet(name: string): string {
  const message = `Hello, ${name}! Welcome to my blog.`;
  console.log(message);
  return message;
}

greet("Yehyeok");
```

정적 빌드 타임에 이 코드 블록이 매력적인 테마로 하이라이팅되길 기대합니다!
