## MODIFIED Requirements

### Requirement: 마크다운 상세 페이지 및 Shiki 하이라이팅 적용
시스템은 각 마크다운 게시글의 상세 페이지를 개별 정적 HTML 파일로 생성해야 하며, 본문 내의 코드 블록에 Shiki 하이라이팅 스타일 클래스가 정상적으로 삽입되어야 한다. 

#### Scenario: 포스트 메타데이터 파싱 및 노출
- **WHEN** 빌드 스크립트가 실행되어 개별 포스트 페이지 정적 파일을 생성할 때
- **THEN** 시스템은 파싱된 Frontmatter 데이터를 기반으로 페이지의 `<head>` 영역에 동적 메타 태그(SEO 및 Open Graph)를 포함하여 HTML을 완성한다.

*(참고: 이외의 하이라이팅 및 렌더링 시나리오는 유지됨)*
