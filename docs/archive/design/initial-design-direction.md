## 톤 & 무드

- **깔끔하고 전문적 + 모던하고 세련된** (GitHub + Toss 사이의 느낌)
- 미니멀하지만 밋밋하지 않은 — 미묘한 애니메이션, 깔끔한 인터랙션
- 여백 넉넉, 타이포그래피 중심의 가독성 우선

## 다크/라이트 모드

- **시스템 설정 기본** + 다크/라이트 모드 선택적 지원
- 사용자의 OS 시스템 테마에 맞추어 자동으로 다크/라이트 모드가 결정됩니다 (`defaultTheme="system"`).
- `localStorage` 기반 다크/라이트 모드 명시적 변경 지원.

## 글 목록 구조

- **피드형** (최신순 나열)이 메인
- 별도 페이지로 주제별/카테고리별 보기 가능
- 태그 기반 필터링 지원

## 페이지 구성

```
/ (홈)
├── 피드형 글 목록 (최신순, 무한 스크롤 or 페이지네이션)
├── 태그/카테고리 필터
└── 간단한 프로필 영역

/posts/[slug] (글 상세)
├── TOC 사이드바 (데스크탑)
├── 스크롤 프로그레스 바
├── Syntax Highlighted 코드 블록 (라이트 모드에서도 Dark Theme 적용) + 복사 버튼
├── 지표 강조 카드 등 MDX 커스텀 컴포넌트 지원
├── 시리즈 내 이전/다음 네비게이션
├── 메타 정보 표시 (조회수, 좋아요, 읽기 시간)
└── 관련 글 추천 (Phase 2)

/about (소개)
└── 프로필, 경력, GitHub 링크 등

/tags/[tag] (태그별 글 목록)
└── 특정 태그의 글만 필터링
```

## 읽기 경험 핵심 요소

| 요소 | 설명 | 우선순위 |
|------|------|----------|
| **TOC 사이드바** | 긴 글에서 현재 위치 파악 + 빠른 네비게이션 | MVP |
| **코드 블록 (Dark Theme)** | 라이트 모드에서도 어두운 배경의 코드 블록 + 복사 버튼 | MVP |
| **MDX 커스텀 컴포넌트** | 지표 강조 카드 등 본문 내 시각적 컴포넌트 삽입 | MVP |
| **읽기 메타 정보** | "읽기 시간 약 N분", 조회수, 좋아요 표시 | MVP |
| **시리즈 네비게이션** | 시리즈 내 이전/다음 글 이동 | MVP |
| **다크/라이트 모드 토글** | 독자 선호에 맞춤 (시스템 모드 기본) | MVP |
| **스크롤 프로그레스 바** | 글 읽기 진행도 표시 | MVP |
| **관련 글 추천** | 글 하단에 유사 주제 글 노출 | Phase 2 |
